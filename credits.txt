DESCRIÇÃO:
Projeto desenvolvido como parte da prova do curso de Desenvolvimento de Sistemas. Inclui interface gráfica (GUI), aplicação web e ferramenta de linha de comando (CLI), integradas com banco de dados.

BIBLIOTECAS UTILIZADAS:
- CustomTKinter, TKinter: para desenvolvimento da GUI
- Python 3.x
- XAMPP (para rodar a aplicação PHP)
- shutil, os, webbrowser, sys (módulos Python usados no script web_app.py)
- hashlib, uuid, sys, random, time, os, mysql.connector, datetime, pymysql (módulos Python usados no script cli_tool.py e gui_app.py)
- PyInstaller (para gerar executáveis, se usado)
- MySQL Connector: para conexão com banco de dados
